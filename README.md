## 📋 About Production-Tree-3D

This software shows a 3D interactive plot of a production tree. It helps you explore production data visually. The tool runs in your web browser through GitHub Pages, so you only need a computer with Windows and an internet connection to use it. No technical or programming skills are required.

The production tree is common in energy fields like oil and gas, mechanical engineering, and geothermal projects. This app helps you see complex production relationships in three dimensions. It can assist engineers and managers by making data easier to understand.

## 🛠️ How to Use the App

- **Zoom In and Out:** Use the mouse wheel or pinch on a touchpad.
- **Rotate View:** Click and hold the left mouse button, then move the mouse around.
- **Select Nodes:** Click on any part of the production tree to get more details.
- **Reset View:** Use the reset button in the interface to return the tree to its original position.
- **Export Image:** Click the export button to save a snapshot of the tree.
  
The interface includes simple labels and tooltips to guide you through the controls. No need for extra settings.

## ⚙️ Customizing Data (Optional)

This app works best with production data files formatted in specific ways. If you want to explore your own data:

The app will translate your file into a 3D tree automatically.
